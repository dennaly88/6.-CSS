CSS

El término "Cascada" se refiere a cómo se aplican las reglas de estilo. Si hay varias reglas de CSS que intentan dar estilo al mismo elemento, el navegador tiene un sistema de prioridades para decidir cuál aplicar. Esto permite que los desarrolladores sobrescriban estilos de forma organizada, asegurando que el diseño final sea el que desean.

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada) y es un lenguaje que se usa para definir el estilo y la presentación de una página web. En pocas palabras, es lo que hace que los sitios web se vean bien, controlando cosas como los colores, las fuentes, los diseños y las animaciones.
